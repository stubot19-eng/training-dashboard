# Bryce's Heavy Lifting Program

## Schedule (45-60 min sessions)
- **Monday:** Push A (Chest focus)
- **Tuesday:** Pull A (Back width)  
- **Wednesday:** Legs A (Quad dominant)
- **Thursday:** REST
- **Friday:** Push B (Shoulder focus)
- **Saturday:** Pull B (Back thickness)
- **Sunday:** Legs B (Posterior chain)

---

## MONDAY — Push A (Chest)
1. **DB Bench Press** — 4×6-8 (heavy, last set to failure)
2. **Incline DB Press** — 3×8-10
3. **Weighted Dips or Machine Press** — 3×8-12
4. **Tricep Pushdowns** — 3×10-12

**Goal:** 65+ lb DBs for 4×8

---

## TUESDAY — Pull A (Width)
1. **Lat Pulldown** — 4×6-8 (heavy, underhand grip last set)
2. **Single-Arm Pulldown** — 3×10-12 each
3. **Cable Rows** — 3×8-10
4. **Barbell Curl** — 3×8-10

**Goal:** 120+ lb pulldowns

---

## WEDNESDAY — Legs A (Quads)
1. **Leg Press** — 4×8-10 (feet low on platform)
2. **Hack Squat or Goblet Squat** — 3×10-12
3. **Leg Extension** — 3×12-15
4. **Calf Raises** — 4×12-15

**Goal:** 250+ lb leg press

---

## THURSDAY — REST
Active recovery, stretch, meal prep.

---

## FRIDAY — Push B (Shoulders)
1. **Seated DB Shoulder Press** — 4×6-8
2. **Incline Machine Press** — 3×10-12
3. **Lateral Raises** — 4×12-15 (rest-pause last set)
4. **Cable Fly** — 3×12-15

**Goal:** 40+ lb shoulder press

---

## SATURDAY — Pull B (Thickness)
1. **Chest-Supported Row** — 4×8-10
2. **Barbell Row** — 3×8-10
3. **Face Pulls** — 4×15-20
4. **Hammer Curls** — 3×10-12

**Goal:** 50+ lb rows

---

## SUNDAY — Legs B (Posterior)
1. **Romanian Deadlift** — 4×8-10
2. **Leg Curl** — 3×12-15
3. **Bulgarian Split Squat** — 3×10-12 each
4. **Plank** — 3×60 sec

**Goal:** 50+ lb RDLs

---

## Rules
- **Rest 2-3 min** between heavy sets
- **Warm up** 2-3 light sets before working weight
- **Log everything** — weight, reps, how it felt
- **Progression:** Add 5 lb when you hit top of rep range

---

*Built for 6'4" 193lb frame — heavy compounds, minimal fluff, maximum results.*
